4 · KEY D3.js FEATURES
D3.js Functions Used
Function / API What it does in this project
Scales
d3.scaleLinear() Maps Study Day → pixel x-position. Domain is updated on every zoom event via rescaleX() to implement pan and zoom.
d3.scaleBand() Distributes patient rows evenly on the y-axis with configurable padding. bandwidth() gives each row's height for sizing event markers.
d3.scaleOrdinal() Assigns Okabe-Ito colors to treatment arms (A / B) and AE grades (1–4). Colorblind-friendly by construction.
Axes
d3.axisBottom() d3.axisTop() Renders bottom Study Day axis and top calendar date axis. Custom tickValues always pins Day 1 and suppresses the spurious Day 0 D3 would otherwise generate. Top axis date is computed independently per patient from their first-dose date.
Data Processing
d3.groups() Splits the flat JSON array (all event types, all patients) into per-patient sub-arrays. The render loop iterates once per group, drawing each patient's layers in isolation.
d3.timeParse() d3.timeFormat() timeParse("%d%b%Y") converts SAS-style date strings (e.g. "01JUN2024") into JS Date objects. timeFormat converts them back for the top axis labels.
.data().join() D3's enter / update / exit binding pattern. Each event layer (EX bars, LB diamonds, AE triangles, PD lines, OS lines) is rendered with one .join() call. Re-runs cleanly when r2d3 re-renders on resize.
Symbols
d3.symbol() symbolTriangle · symbolDiamond Generates SVG <path> strings for AE markers (▲ triangle, size scales with grade 1–4) and Lab CS markers (♦ diamond). No custom SVG path math needed.
Interaction (Interactive Timeline only)
d3.zoom() transform.rescaleX() Captures scroll-wheel and trackpad pinch gestures. On each zoom event, rescaleX(xScale) returns a new scale with the updated domain — all axes and markers re-render against the new scale. Powers ★2 Semantic Zoom.
d3.zoomTransform() Reads the current zoom level k at any moment (e.g. on hover). Used to decide whether to show Grade 1–2 AE labels: visible only when k ≥ 3.
Rendering
SVG clipPath A rectangular clip mask attached to the plot area. Prevents event markers from bleeding outside panel boundaries when panning or zooming. Uses a random suffix on the ID to avoid collision on r2d3 re-render.
selection.attr("opacity") The single mechanism behind three features: ★2 fades Grade 1–2 at low zoom; ★3 dims all layers except the hovered patient to 8%; ★4 dims deselected event-class layers. No CSS transitions — direct attribute set for instant response.
